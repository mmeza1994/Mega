# Proyecto Mega con PHP y PHPTAL

## Estructura del Proyecto

- `css/`: Archivos CSS
- `img/`: Imágenes
- `templates/`: Plantillas PHPTAL
- `sql/`: Scripts SQL para crear y llenar la base de datos
- `vendor/`: Dependencias de Composer
- `index.php`, `insert.php`, `new_journalist.php`, `edit_article.php`, `delete_article.php`: Archivos PHP del proyecto

## Instalación

### Requisitos

- PHP 7.4 o superior
- Composer
- Servidor web (XAMPP, WAMP, etc.)
- Base de datos MySQL
